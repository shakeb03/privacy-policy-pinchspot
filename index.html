The App does not collect, store, or process any personal data from its users. Uploaded images are temporarily processed locally for functionality but are not saved, shared, or transmitted to any server.

The App integrates with the Pinterest API to retrieve content related to user inputs (e.g., image areas selected). This API access is governed by Pinterest's API Terms of Use. No user information is shared with Pinterest or any third parties.

Data Security

As no personal data is collected or stored by the App, there is no risk of data breach. The application processes images locally on your device for recommendations.

The App uses Pinterest’s API solely for retrieving and displaying relevant content. It complies with Pinterest’s API usage guidelines.
